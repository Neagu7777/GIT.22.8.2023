 1 уровень сложности: 1.Как отказаться от изменений в файле (файлах) находящийся в рабочем каталоге (work directory)?
2.Как вернуть файл из индекса в рабочий каталог?
3.Как отменить ошибочный commit ? - Перечислить все возможные варианты.
4.Как исправить коммит отправленный на сервер GitHub?
5.Как создать новую ветку и переключится в нее?

В ответах на вопросы необходимо указать конкретные команда git.

1) a) git checkout . ( если во всех файлов) git checkout имя файла ( если в один файл) 
b) git restore (name file )

2) git restore --staged  имя файла 
3) git commit --amend -m " "   если надо поменять комментарий 
Если нужно добавить новый файл то добавляем его в индекс и тоже git commit --amend -m " " 
git reset HEAD ~ сколько комита 
4) git revert HEAD 
Git push -- force ( но лучше не так ) 
5) git branch имя ветки 
Git checkout для смены ветки но можно добавить -b и имя ветки создаём и переключяемся к ней
